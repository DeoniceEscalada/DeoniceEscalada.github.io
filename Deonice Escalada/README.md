# Deonice Escalada Portfolio

Welcome to the Deonice Escalada portfolio project! This project showcases my work as a creative professional, including custom illustrations and design projects.

## Project Structure

```
Deonice Escalada
├── templates
│   └── index.html          # Main HTML structure of the portfolio website
├── src
│   ├── styles
│   │   └── main.css        # CSS styles for the website
│   ├── scripts
│   │   └── main.js         # JavaScript functionality for the website
│   └── data
│       └── projects.json    # JSON data for projects
├── package.json            # npm configuration file
├── .gitignore              # Files and directories to ignore by Git
└── README.md               # Project documentation
```

## Setup Instructions

1. Clone the repository to your local machine:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd Deonice Escalada
   ```

3. Install the necessary dependencies:
   ```
   npm install
   ```

4. Open `templates/index.html` in your browser to view the portfolio.
